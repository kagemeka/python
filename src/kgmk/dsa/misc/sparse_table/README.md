# Sparse Table 


# keywords
- get range query
- idempotent
- associative property
- semi-group
- $O(N\log{N})$ preprocess.
- answer for query with $O(1)$.


# summary
- idempotent semi-group example
  - $(\mathbb{Z}, \min)$
  - $(\mathbb{Z}, \max)$
  - $(\mathbb{Z}, \gcd)$
  - $(\mathbb{Z}, \text{lcm})$
